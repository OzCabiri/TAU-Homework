SHELLCODE:
1. To set EAX=11, we can't use 'MOV EAX, 0x0B' because that translate to 'B8 0B 00 00 00'.
    Solution: XOR EAX with itself the zero it ('31 C0') and add 0x0B to it ('83 C0 0B').
    
2. To add \0 at the end of EBX, we can't use MOV BYTE PTR [EBX+7], 0x00 (obviously contains a zero).
    Solution: XOR ECX with itself and MOV [EBX+7], ECX (similar concept to before).
    
3. Can't MOV EDX 0x00 because of zero, so we XOR it (envp can be NULL).

PYTHON CODE:
We will choose a password that contains our shellcode and overrides the return address.
So our password will end with SHELLCODE+RETURN_ADDRESS.
We will pad the end of the password with nop as instrcuted in the assignment.

buffer_start is at 0xbfffe069, according to the gdb analysis (where we start having 0x61 repeating), so this will be our return address.

After assembling, the file ends with 0, so we will replace with another nop.

Our password looks like this: SHELLCODE + nop_slide + return_address (start of shell code)
